# NetworkProxy

本项目包含一系列用于网络代理工具（如 Loon 等）的脚本，主要用于解锁部分 App 的会员功能。

## 支持的 App 及功能

| App        | 功能                         | 支持的 App 版本                           |
| :--------- | :--------------------------- | :---------------------------------------- |
| Foodie     | 会员解锁                     | ≤ 7.4.40                                  |
| 美图秀秀   | 会员解锁                     | ≤ 12.9.0                                  |
| 美颜相机   | 会员解锁                     | ≤ 11.4.00                                 |
| 美图全家桶 | 美颜相机 & 美图秀秀 会员解锁 | 美颜相机 ≤ 11.4.00 <br> 美图秀秀 ≤ 12.9.0 |
| Pillow     | 会员解锁                     | ≤ 6.3                                     |
